# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025)

## Structure

Each day's solution is in its own folder:
- `dayX/dayX.py` - Python solution for day X
- `dayX/input.txt` - Puzzle input
- `dayX/sample.txt` - Sample input from problem description

## Running Solutions

```bash
cd 2025/dayX
python dayX.py
```
